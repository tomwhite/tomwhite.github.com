--- layout: post title: L-Systems in SVG and Javascript date: '2007-11-18T15:06:00.000Z' author: Tom White tags: modified_time: '2007-11-18T15:12:09.733Z' thumbnail: http://2.bp.blogspot.com/_IhqEHw4_Ick/R0BWCCPnAHI/AAAAAAAAABQ/2K-CXLbq0uE/s72-c/bushes.png blogger_id: tag:blogger.com,1999:blog-8898949683610477251.post-724528594416527321 --- While waiting for broadband to be enabled, I spent some time playing with L-systems. A few years ago I wrote some Java code to render L-systems, using the Java 2D API. The seemed like a good fit for SVG and Javascript, so I brushed up on both and got going.

The result is that you can define your L-system as a Javascript map, then turn it into an SVG Path. You can then do all kinds of SVG magic, such as transforming it, changing the colour, line style, etc.
