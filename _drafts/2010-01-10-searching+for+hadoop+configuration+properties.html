A nice feature of O'Reilly's Safari Books Online is the powerful search facility. I found this useful recently when searching for documention of a Hadoop configuration property in my book, Hadoop: The Definitive Guide, since it returns the page numbers that the property is mentioned on.

For example, a search for io.sort.mb lists the pages where this configuration option appears, as well as a snippet of context. This works even if you are not a Safari subscriber. In this case the main entry is on page 167, which, in case you don't have the book to hand and are a Safari subscriber, has the simple URL http://my.safaribooksonline.com/9780596521974/167 (9780596521974 is the book's ISBN).

For basic information about every Hadoop property (not all are covered in the book), you can consult the "default" XML file that defines the properties and their default values. A short textual description is provided for each property. For releases before 0.20.0, the configuration was in a single file, hadoop-default.xml, found in the conf directory of the distribution. This file is also rendered in HTML form in the documentation; here's the one for Hadoop 0.18.3, for example. You can even link to a particular property by adding a URL fragment, such as #io.sort.mb.

For release 0.20.0 and later, the default files have moved. The configuration file has been split into three, one for each of Hadoop Common, HDFS, and MapReduce. These files, core-default.xml, hdfs-default.xml, mapred-default.xml, are no longer in the conf directory. They are however in the src directory, in the core, hdfs, and mapred directories (respectively).
